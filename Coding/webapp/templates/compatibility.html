How is this score calculated? ▸
1
Raw stat → formula → one number per metric
Each metric in the role profile has a computation formula applied to the player's raw SofaScore stats. There are three formula types:
2
Raw value → 0–100 metric score
Two scoring paths depending on the formula:
Percentage / rate output
score = raw value (already 0–100)
e.g. "Pass accuracy %" — 73.2% → score 73.2
Per-90 counting stat
score = (raw / ref_max) × 100, clamped to 100
Each stat has an elite benchmark as ref_max (e.g. goals/90 ref_max = 1.2, key passes/90 = 4.0). A player at the benchmark scores 100.
For metrics where lower is better (e.g. fouls/90), the score is inverted: score = 100 − raw_score.
